J'observe/J'apprends les oiseaux de Toulouse

Malette pédagogique pour apprendre à identifier et observer les principaux oiseaux de Toulouse avec des enfants de 6 à 10 ans.

Cette malette est consitutée de :
- 10 carnets "J'observe les oiseaux de Toulouse"
- 10 carnets "J'apprends les oiseaux de Toulouse"
- 68 transparents qui constituent 17 oiseaux toulousains
- un numéro de malette et un code d'accès pour se connecter sur le site relié : oiseaux-toulouse.fr

Les carnets "J'observe les oiseaux de Toulouse" guide la prise de note lors d'une première sortie en ville, dans les parcs ou jardins pour observer les oiseaux avec les enfants.

Ensuite, les carnets "J'apprends les oiseaux de Toulouse" accompagnés des transparents servent à identifier les principales caractéristiques des oiseaux observés et/ou observables.
Le jeu de transparents peut tant être manipulé par les enfants (pour reconstituer un oiseau) que par l'animateur comme image de référence ou pour reconstruire l'oiseau au fur et a mesure que les différentes caractéristiques d'un oiseaux sont énoncées par les enfants.

Une seconde sortie suite à cette partie d'apprentissage permettrait de tester ces nouvelles capacités d'identification avec de véritables oiseaux.
Le site internet aurait deux fonctions principales :
- réimprimer d'autres carnets pour en faire profiter le maximum d'enfants
- proposer une banques de références pour aider les animateurs à préparer leur activité.
